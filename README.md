# Web-Technologies-Lab-07
# Task Requirements:
A city administration wants a simple weather monitoring dashboard that retrieves and displays the current weather forecast for different cities using a public weather API. The system should dynamically show the data on a webpage so that users can quickly view weather conditions. 
# 1. Data Retrieval:
Use fetch API to retrieve weather data for multiple cities from a provided weather API endpoint.
# 2. Data Presentation:
Dynamically create an html list that displays atleast the following fields.
1. City Name
2. Temperature
3. Weather Condition (Sunny, Cloudy, Rainy)
# 3. Error Handling:
Implement proper error handling if API fails to respond display a red message unable to load weather data.
# 4. User Interaction:
Add a "Refresh Weather" button that allows the user to update weather information without reloading the page.
# Performed the above tasks using techniques:
# 1. JavaScript
# 2. TypeScript
# 3. Angular Js
